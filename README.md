# 🎨 CodeCraft Academy - Interactive Course Platform

A **modern, interactive course platform** featuring industry-level AI + Full Stack mentorship with cutting-edge UI/UX design and animations.

## ✨ Features

### 🎯 Core Functionality
- **Home** – Hero with particle effects, offerings, animated stats, CTA
- **About** – Company story, values, interactive timeline
- **Mentors** – Industry expert profiles with hover effects
- **Learners** – Testimonials and success stories
- **Courses** – Course listings with advanced filtering
- **Course Details** – Curriculum, outcomes, tech stack
- **Contact** – Interactive contact form

### 🎨 Advanced UI Features
- **🌟 Particle Effects** - Floating animated particles
- **🎭 Dark/Light Theme Toggle** - Smooth theme switching
- **🧲 Magnetic Buttons** - Proximity-based interactions
- **📜 Text Reveal Animations** - Character-by-character animations
- **🎪 3D Card Transforms** - Interactive 3D hover effects
- **🌊 Scroll Animations** - Parallax and reveal effects
- **💫 Cursor Trail Effects** - Custom cursor with glowing trail
- **🎨 Morphing Backgrounds** - Animated gradient shapes
- **⚡ Micro-interactions** - Button ripples, loading states

## 🛠 Tech Stack

### Frontend
- **React 19** with TypeScript
- **Framer Motion** for advanced animations
- **React Router** for navigation
- **CSS Variables** for theming
- **Custom Hooks** for scroll animations

### Backend
- **.NET Core** Web API
- **Entity Framework** Core
- **SQL Server** integration ready

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm start
```
Opens at [http://localhost:3000](http://localhost:3000)

### Backend
```bash
cd backend
dotnet restore
dotnet run
```
API runs at [http://localhost:5000](http://localhost:5000)
